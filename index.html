--- layout: page title: "Home" --- Hi! This is my first personal public webpage... ## Images Upload an image to the *assets* folder and embed it with `![title](/assets/name.jpg))`. Keep in mind that the path needs to be adjusted if Jekyll is run inside a subfolder.
